# Creación de un agente para análisis de datos

En este laboratorio crearás un agente de análisis de datos en **Database Private Agent Factory (DPAF)**. El agente permite consultar los datos cargados en la base de datos mediante lenguaje natural y obtener respuestas, tablas y visualizaciones. Puedes aplicar este proceso a archivos CSV de distintos tipos y estructuras de datos.


## Paso 1: Configuración de la fuente de datos

Ingresa a **Database Private Agent Factory (DPAF)**. En el panel de navegación izquierdo, selecciona la opción **Data Source**.

Crea un nuevo Data Source de tipo **Database** completando el formulario con los siguientes campos:

- **Source type:** selecciona **Database**.
- **Source name:** escribe `datos`. Usaremos este nombre para identificar y seleccionar la fuente durante la creación del agente.
- **Description:** escribe `Fuente de datos para el agente`.
- **Connection type:** selecciona **Wallet**.
- **Wallet:** carga el archivo `.zip` de la wallet descargada previamente. Puedes arrastrarlo al área **Drag and Drop** o seleccionarlo desde tu equipo.
- **Database alias**: ..._medium. Selecciona el alias que termina en medium.
- **Usuario:** El usuario del esquema.
- **Contraseña:** la contraseña que definiste al crear la base de datos

![Formulario de configuración del Data Source](./images/dpaf_image12.png)

Una vez completado el formulario, haz clic en **Test Connection** para validar que la conexión sea exitosa. Si la prueba es exitosa, presiona el botón **Add Database Source** para guardar la fuente de datos.

### Verificación de la fuente de datos

Si la configuración fue correcta, el nuevo Data Source aparecerá listado en el panel de **Data Source** con el nombre `datos`.

## Paso 2: Creación del agente de análisis

Vuelve al menú de navegación izquierdo y haz clic en **Data Analysis Agents**. Luego presiona el botón **Create Agent** para iniciar la configuración del agente.

![Panel de Data Analysis Agents](./images/dpaf_image13.png)

![Botón Create Agent](./images/dpaf_image14.png)

### Selección de la base de datos

En el formulario de creación del agente, selecciona la fuente de datos `datos`, que acabas de configurar.

### Selección de tablas

Utiliza la barra de búsqueda para encontrar y seleccionar una tabla que el agente utilizará. El nombre de la tabla corresponde al nombre de uno de los archivos CSV cargados previamente (sin la extensión `.csv`).

![Barra de búsqueda de tablas](./images/dpaf_image15.png)

![Resultado de búsqueda — tabla "datos"](./images/dpaf_image16.png)

> **Ejemplo:** si el archivo se llamaba `datos.csv`, la tabla se llamará `datos`.

Una vez seleccionadas las tablas, haz clic en el botón **Add New Source** para confirmar la selección y avanzar al siguiente paso.

![Botón Add New Source](./images/dpaf_image17.png)

### Revisión de la configuración del agente

En este paso puedes proporcionar instrucciones y contexto para orientar el comportamiento del agente según los datos cargados y el tipo de análisis que necesitas realizar.

Revisa el resumen de configuración del agente. Verifica que la base de datos y las tablas seleccionadas sean correctas. Si todo está en orden, haz clic en **Next** para continuar.

![Resumen de configuración del agente](./images/image%2048.png)

![Botón Next](./images/dpaf_image20.png)

### Publicación del agente

Si la configuración está completa y validada, presiona el botón **Publish Agent** para publicar el agente y dejarlo disponible para su uso.

![Botón Publish Agent](./images/dpaf_image19.png)

### Acceso al agente publicado

Una vez publicado, el agente aparecerá listado en el panel de **Data Analysis Agents** del menú izquierdo. Haz clic en **Open Agent** para acceder a él y comenzar a utilizarlo.

## Paso 3: Uso del agente para análisis de datos

### Apertura del agente

Al hacer clic en **Open Agent**, se abrirá el panel principal del agente, desde donde puedes interactuar con los datos cargados.

![Panel principal del agente](./images/dpaf_image21.png)

### Exploración automática de los datos

Haz clic en el botón **Execute Exploration** para que el agente analice automáticamente los datos. Según los tipos de datos detectados en cada columna, el agente generará distintas visualizaciones y gráficas que te permitirán entender la distribución y estructura del dataset.

![Exploración de datos con gráficas](./images/dpaf_image22.png)

### Consulta de datos en lenguaje natural

Puedes hacerle preguntas al agente directamente en lenguaje natural. El agente interpretará tu pregunta, generará una consulta SQL sobre la base de datos y te devolverá la respuesta.

Formula preguntas relacionadas con los datos cargados para explorar la información disponible.

### Visualización de la consulta SQL generada

Para ver la consulta SQL que el agente ejecutó para responder tu pregunta, haz clic en el botón **SQL**. Esto te permite auditar y entender cómo el agente traduce las preguntas a consultas sobre la base de datos.
