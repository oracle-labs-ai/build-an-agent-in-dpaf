# Carga de los datos

En esta guía cargarás uno o varios archivos CSV en la base de datos para que estén disponibles al crear un agente de análisis de datos. Puedes utilizar archivos con distintas estructuras y tipos de información.

## Paso 1: Abrir Data Load

En la consola web de OCI, navega hasta la pantalla principal de la base de datos. Haz clic en **Database Actions** y, dentro del menú, selecciona **Data Load**.

![Botón Data en Database Actions](./images/dpaf_image3.png)

Se abrirá un panel con el botón **Load Data**. Haz clic en él para continuar.

![Botón Load Data](./images/dpaf_image4.png)

## Paso 2: Seleccionar los archivos CSV

En el panel de carga puedes arrastrar y soltar tus archivos CSV, o hacer clic en **Select Files** para buscarlos en tu equipo. Puedes cargar uno o varios archivos según las tablas que quieras utilizar posteriormente.

![Panel de arrastrar y soltar archivos](./images/dpaf_image6.png)

> **Nota:** Los archivos CSV pueden estar separados por `,` (coma) o por `;` (punto y coma). La plataforma intentará detectarlo automáticamente.

## Paso 3: Revisar la configuración de los archivos

Una vez cargado un archivo, aparecerá un bloque con su nombre. Este nombre corresponderá al de la tabla que se creará en la base de datos. Verás la advertencia **Review Settings**.

![Bloque de detección del dataset](./images/dpaf_image7.png)

Haz clic en **Review Settings** y verifica lo siguiente:

- Los datos se cargaron correctamente.
- Los tipos de datos de cada columna se detectaron de forma correcta.
- El separador del CSV fue identificado correctamente.

Corrige cualquier ajuste necesario. Cuando todo esté en orden, la advertencia **Review Settings** desaparecerá. Si no necesitas hacer cambios, presiona **Close**.

![Botón Close](./images/image%2047.png)

## Paso 4: Iniciar la carga

Con la configuración validada, haz clic en **Start** para iniciar la transferencia de datos a la base de datos.

![Botón Start](./images/dpaf_image9.png)

## Paso 5: Verificar la carga

Al finalizar, el bloque mostrará el nombre del archivo junto con el **número de columnas y filas** cargadas. Confirma que estos valores coincidan con los de tu archivo original.

![Resumen de carga exitosa](./images/dpaf_image10.png)

## Paso 6: Cerrar el panel

Una vez verificada la carga, haz clic en **Close** para finalizar el proceso.

![Botón Close](./images/dpaf_image8.png)

Los datos cargados quedarán disponibles para configurar la fuente de datos y crear un agente de análisis en el siguiente laboratorio.
